# Как сложить числа? :fu:
## С помощью оператора +
```python
num1 = 5
num2 = 10
res = num1 + num2
print(res)
#result
15
```
## С помощью метода fsum модуля math
```python
import math

lst = [2, 4, 6]
print(math.fsum(lst))
#result
12.0
```
